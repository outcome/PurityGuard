# PurityGuard

PurityGuard is an Android DNS-based protection app focused on simple UX and reliable blocking.

## Highlights
- Local VPN DNS enforcement
- Multiple filtering providers (Cloudflare, CleanBrowsing, OpenDNS)
- Custom blocked-page experience with motivational content
- IPv6-aware DNS handling
